    Three main differences between Web1 , Web2 , Web3 :

    מבחינה טכנית

    סגור לעריכה חיצונית של דפי אינטרנט. המידע אינו דינאמי והוא לקריאה בלבד הוא מוזן על ידי בוני האתרים Web 1.0

    פתוח גם לקריאה וגם לכתיבה של משתמשים המידע משותף בין משתמשים שונים Web 2.0

    פתוח לקריאה ולכתיבה. בנוסף, ניתן לבצע עריכת דפי אינטרנט. Web 3.0

    מבחינה חברתית

    אין תקשורת בין משתמשים. המשתמש רק צופה בתכני האינטרנט Web 1.0

    מאפשר למשתמשים לתקשר אחד עם השני בצורה יותר חופשית. הוא מעודד שיתוף מידע וקשרים Web 2.0

    מספק שירות אינטראקטיבי בו יש אינטראקציה בין מחשב למחשב Web 3.0

    מבחינת אחזור והפקת מידע

    המידע שמור בספריות שמהן מבצעים את איחזור המידע Web 1.0

    המידע מופק ע"י תגיות Web 2.0

    המידע מופק ע"י למידת מכונה Web 3.0
