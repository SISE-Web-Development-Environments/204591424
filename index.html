About Me


קצת על חמי וחמותי דניאל ויהודית צרור...

דניאל הוא בן 75 ויהודית בת 68. הם ילידי טוניס. הכירו בארץ ישראל והתחתנו בגיל 20.
לדניאל ויהודית יש 5 ילדים - איילה שמעוני, עוזי צרור, איריס לוי, לילך מאיר ובעלי הראל צרור.
יש להם 28 נכדים ואתמול ב6/4/2020 נולד להם במזל טוב ובשעה טובה נין ראשון.
דניאל ויהודית הם אנשי משפחה ופנסיונרים. הם אנשים בעלי נתינה ענקית ומאוד אוהבים להיות בחיק המשפחה.
בנוסף, הם ממש אוהבים את הטבע ולעשות בזמנם הפנוי טיולים.
לאור המצב של הקורונה וגילם המבוגר הם כרגע בבידוד ולראשונה יעשו את החג לבד ולא בחיק המשפחה המורחבת
הדבר מאוד מצער אותם במיוחד שהם אנשים שאוהבים לתת ולתרום לאחרים ולעשות למען משפחתם
עם זאת, המשפחה הקרובה מבקרת את ההורים דרך גדר הגינה מדי פעם, שומרים על קשר טלפוני ומקווים שהמצב יעבור במהירות.

Questions & Answers
Three main differences between Web1 , Web2 , Web3 :
מבחינה טכנית

סגור לעריכה חיצונית של דפי אינטרנט. המידע אינו דינאמי והוא לקריאה בלבד הוא מוזן על ידי בוני האתרים Web 1.0

פתוח גם לקריאה וגם לכתיבה של משתמשים המידע משותף בין משתמשים שונים Web 2.0

פתוח לקריאה ולכתיבה. בנוסף, ניתן לבצע עריכת דפי אינטרנט. Web 3.0

מבחינה חברתית

אין תקשורת בין משתמשים. המשתמש רק צופה בתכני האינטרנט Web 1.0

מאפשר למשתמשים לתקשר אחד עם השני בצורה יותר חופשית. הוא מעודד שיתוף מידע וקשרים Web 2.0

מספק שירות אינטראקטיבי בו יש אינטראקציה בין מחשב למחשב Web 3.0

מבחינת אחזור והפקת מידע

המידע שמור בספריות שמהן מבצעים את איחזור המידע Web 1.0

המידע מופק ע"י תגיות Web 2.0

המידע מופק ע"י למידת מכונה Web 3.0
